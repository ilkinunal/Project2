***Web & HTML for Beginners

Definitions

HTML stands for Hyper Text Markup Language which is the main document type of the web, invented in 1990s.
How the Web Works?
When a user opens a web browser and writes www.udacity.com, the computer sends an HTTP request to a server.
HTML
HTML has "Text content" (what you see), "markup" (what it looks), "references to other documents" (images, videos, etc), "links to other pages".
